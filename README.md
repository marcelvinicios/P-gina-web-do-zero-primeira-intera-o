# 🚀 Tech Blog

Um blog de tecnologia focado em criação de conteúdo e lógica de interações do usuário.

---

## 🛠️ Funcionalidades

* **✍️ Criação de posts:** Interface para publicação de novos artigos sobre tecnologia.
* **👍 Botões de "curtir" e "não curtir":** Sistema de reações para engajamento nas publicações.
* **🧠 Controle de interações:** Implementação de regras e limites para as reações (ex: evitar múltiplos votos).
